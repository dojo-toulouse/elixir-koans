elixir-koans
============

Install
-------

```bash
  $ git clone git@github.com:dojo-toulouse/elixir-koans.git
  $ cd elixir-koans
```

Compile

```bash
  $ elixirc utils/Koans.ex
```

Launch
-------

**You can safely ignore warnings**

```bash
  $ elixir path_to_enlightenment.exs
```

or

```bash
  $ path_to_enlightenment.exs
```

To medidate on a single theme, run:
```bash
  $ elixir about_*.exs
```

or

```bash
  $ about_*.exs
```
